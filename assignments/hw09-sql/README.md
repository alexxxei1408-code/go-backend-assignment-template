# Домашнее задание 9: Работа с SQL в Go

## Цель

Научиться выполнять SQL-запросы в Go, используя стандартный пакет `database/sql` и работать с контекстом.

## Краткое описание задания

Вам нужно реализовать функции для работы с таблицей пользователей в базе данных.
Структура `User` и сигнатуры функций заданы в `main.go`.

Необходимо:

1. Реализовать функцию `InsertUser`:
   - добавляет нового пользователя в таблицу `users`
   - рекомендуется использовать SQL: `INSERT INTO users (id, name, email, active) VALUES (?, ?, ?, ?)`
   - должна использовать `ExecContext` с переданным контекстом

2. Реализовать функцию `GetActiveUsers`:
   - возвращает всех активных пользователей, отсортированных по возрастанию `ID`
   - рекомендуется использовать SQL: `SELECT id, name, email, active FROM users WHERE active = TRUE ORDER BY id`
   - должна использовать `QueryContext` и собирать результаты в слайс

3. Реализовать функцию `DeactivateUser`:
   - помечает пользователя как неактивного по его `ID`
   - рекомендуется использовать SQL: `UPDATE users SET active = FALSE WHERE id = ?`
   - должна использовать `ExecContext` с переданным контекстом
