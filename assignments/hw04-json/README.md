# Домашнее задание 4: Работа с JSON

## Цель

Научиться сериализовать и десериализовать данные в формате JSON, работать с JSON-тегами и фильтровать данные.

## Краткое описание задания

Вам нужно реализовать функции для работы с JSON-представлением пользователей.
Структура `User` и сигнатуры функций заданы в `main.go`.

Необходимо:

1. Реализовать функцию `DecodeUsers`, которая:
   - читает из `io.Reader` JSON-массив объектов пользователей
   - игнорирует неизвестные поля в JSON
   - возвращает ошибку при некорректном JSON или если входные данные не являются массивом

2. Реализовать функцию `EncodeActiveUsers`, которая:
   - записывает в `io.Writer` JSON-массив только активных пользователей (с `Active == true`)
   - сортирует пользователей по возрастанию `ID`
   - форматирование JSON (пробелы, переносы строк) значения не имеет
