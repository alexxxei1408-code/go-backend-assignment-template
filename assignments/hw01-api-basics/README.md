# Домашнее задание 1: Базовый HTTP API

## Цель

Создать HTTP-сервер с одним эндпоинтом `/api/hello`.

## Требования

1. Сервер должен слушать на порту `8080`.
2. При `GET`-запросе к `/api/hello?name=Имя` возвращать JSON:

   ```json
   {"message": "Hello, Имя!"}
   ```

3. Если параметр `name` не указан, использовать `"World"`.
4. При `POST`-запросе к `/api/hello` возвращать ошибку `405 Method Not Allowed`.

## Стартовая точка

Файл `main.go` содержит функцию `HandleHello` — реализуйте её.

## Запуск тестов

```bash
cd assignments/hw01-api-basics
go test -v
```
