# Домашнее задание 10: Комбинированная задача

## Цель

Применить полученные знания для реализации комплексного решения, объединяющего HTTP-хэндлеры, работу с SQL и конкурентность.

## Краткое описание задания

Вам нужно реализовать HTTP-эндпоинт для получения статистики по пользователям из базы данных.
Структура `Stats` и сигнатуры функций заданы в `main.go`.

Необходимо:

1. Реализовать функцию `LoadUserStats`:
   - загружает статистику по активным и неактивным пользователям из БД
   - необходимо выполнить два запроса:
     - `SELECT COUNT(*) FROM users WHERE active = TRUE`
     - `SELECT COUNT(*) FROM users WHERE active = FALSE`
   - рекомендуется выполнять запросы конкурентно с синхронизацией через `sync.WaitGroup`
   - ошибки можно передавать через каналы

2. Реализовать функцию `StatsHandler`:
   - создаёт `http.Handler`, который отвечает на `GET /stats` JSON-объектом `Stats`
   - для обработки запроса нужно использовать `LoadUserStats` с контекстом запроса
   - при успешной загрузке возвращает `200 OK` и JSON-тело
   - при ошибке БД возвращает `500 Internal Server Error`
   - `Content-Type` ответа должен начинаться с `"application/json"`
