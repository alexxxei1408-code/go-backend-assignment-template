# Домашнее задание 2: Интерфейсы и структуры

## Цель

Научиться проектировать интерфейсы, реализовывать методы для структур и работать с полиморфизмом в Go.

## Краткое описание задания

Вам нужно реализовать систему управления банковскими депозитами с различными типами начисления процентов.
Интерфейс `Deposit`, структуры депозитов и методы заданы в `main.go`.

Необходимо:

1. Реализовать методы для `SimpleDeposit` (простой процент):
   - `Balance()` — текущий баланс с учётом начисленных процентов
   - `Interest()` — сумма начисленных процентов
   - `MaturityValue()` — итоговая сумма к концу срока

2. Реализовать методы для `CompoundDeposit` (сложный процент):
   - `Balance()` — баланс с учётом капитализации процентов
   - `Interest()` — сумма начисленных процентов
   - `MaturityValue()` — итоговая сумма к концу срока

3. Реализовать методы для `FixedDeposit` (фиксированный доход):
   - `Balance()` — текущий баланс (равен начальному вкладу)
   - `Interest()` — фиксированная сумма процентов
   - `MaturityValue()` — начальный вклад плюс фиксированные проценты

4. Реализовать методы для `DepositPortfolio`:
   - `Add()` — добавление депозита в портфель
   - `TotalBalance()` — суммарный баланс всех депозитов
   - `MostProfitable()` — депозит с максимальной доходностью (по `Interest()`)
