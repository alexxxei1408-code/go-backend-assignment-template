# Домашнее задание 1: Слайсы и map

## Цель

Научиться эффективно агрегировать данные с помощью слайсов и ассоциативных массивов (map),
а также работать с сортировкой и расчётом производных метрик.

## Краткое описание задания

Вам нужно реализовать агрегирующие функции для анализа покупок пользователей.
Структуры `Purchase` и `UserStats`, а также сигнатуры функций заданы в `main.go`.

Необходимо:

1. Реализовать функцию `TopUsersBySpending`, которая:
   - группирует покупки по пользователям;
   - считает суммарные траты и количество уникальных категорий;
   - возвращает не более `topN` пользователей,
   отсортированных по суммарным тратам (по убыванию), а при равенстве — по идентификатору пользователя (по возрастанию);
   - не изменяет входной слайс.
2. Реализовать функцию `CategoryTotals`, которая возвращает сумму трат по каждой категории
   в виде `map[string]int64`. При отсутствии покупок должна возвращаться пустая, но не `nil` map.
