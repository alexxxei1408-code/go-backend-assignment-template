# Домашнее задание 7: Конкурентность 2. Паттерны

## Цель

Научиться применять паттерны конкурентного программирования: worker pool, fan-in и pipeline.

## Краткое описание задания

Вам нужно реализовать пул воркеров для параллельной обработки задач и функцию объединения каналов.
Структуры `Job`, `Result` и сигнатуры функций заданы в `main.go`.

Необходимо:

1. Реализовать функцию `RunWorkerPool`:
   - обрабатывает список задач с помощью пула из фиксированного числа воркеров
   - каждый `Job` должен быть обработан ровно один раз
   - для вычисления результата вызывается функция `fn` от поля `Input`
   - возвращаемый слайс результатов должен быть отсортирован по `JobID` по возрастанию
   - при `workers <= 0` следует использовать один воркер
   - рекомендуется использовать паттерн worker pool: канал заданий, фиксированное число воркеров и канал результатов

2. Реализовать функцию `FanIn`:
   - объединяет несколько входных каналов в один выходной
   - должна запускать отдельные горутины для чтения из каждого входного канала
   - выходной канал должен быть закрыт после закрытия всех входных
   - порядок элементов в выходном канале не регламентируется
