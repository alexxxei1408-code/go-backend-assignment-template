# Домашнее задание 3: Работа со строками и strings.Builder

## Цель

Научиться эффективно работать со строками в Go, используя `strings.Builder` для минимизации аллокаций памяти.

## Краткое описание задания

Вам нужно реализовать функцию форматирования логов приложения, которая эффективно собирает строку из множества записей.
Структура `LogRecord` и сигнатура функции заданы в `main.go`.

Необходимо:

1. Реализовать функцию `FormatLog`, которая:
   - форматирует каждую запись в виде `"<LEVEL>: <Message>\n"`, где `LEVEL` — значение поля `Level` в верхнем регистре
   - объединяет все записи в одну строку
   - при пустом слайсе возвращает пустую строку
   - для эффективности использует `strings.Builder` вместо конкатенации строк

cd assignments/hw03-strings-builder
