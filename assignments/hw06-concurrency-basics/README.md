# Домашнее задание 6: Конкурентность 1. Основы

## Цель

Научиться создавать потокобезопасные структуры данных с использованием мьютексов и реализовывать простые конкурентные вычисления.

## Краткое описание задания

Вам нужно реализовать потокобезопасный счётчик и функцию параллельного суммирования.
Структура `SafeCounter` и сигнатура функции заданы в `main.go`.

Необходимо:

1. Реализовать `SafeCounter`:
   - должен быть безопасен для использования из нескольких горутин
   - рекомендуется использовать `map[string]int` и `sync.Mutex` или `sync.RWMutex`
   - методы `Inc`, `Value` и `Total` должны быть потокобезопасными

2. Реализовать функцию `SumConcurrently`:
   - считает сумму элементов слайса с использованием нескольких горутин
   - количество воркеров задаётся параметром `workers`
   - при `workers <= 0` следует использовать один воркер
   - для синхронизации рекомендуется использовать каналы или `sync.WaitGroup`
