# Домашнее задание 5: Работа с io.Reader и io.Writer

## Цель

Научиться создавать обёртки над стандартными интерфейсами `io.Reader` и `io.Writer` для расширения их функциональности.

## Краткое описание задания

Вам нужно реализовать два типа-обёртки для работы с потоками данных.
Структуры и сигнатуры функций заданы в `main.go`.

Необходимо:

1. Реализовать `LimitReader`:
   - ограничивает количество байт, читаемых из базового `io.Reader`
   - после прочтения `N` байт все последующие вызовы `Read` должны возвращать `0, io.EOF`
   - должен корректно обрабатывать поэтапные чтения

2. Реализовать `CountingWriter`:
   - оборачивает `io.Writer` и считает количество успешно записанных байт
   - поле `N` должно накапливать сумму всех фактически записанных байт
   - должен корректно обрабатывать множественные вызовы `Write`
